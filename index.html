Select Projects

Data-Driven Development of Environmental Sensors

Developed MATLAB-based software for sensor data acquisition, cleaning, and statistical analysis to measure drinking water contamination. Implemented a full-factorial statistical optimization to enhance sensor sensitivity and dynamic range.

Extending the Theory of Deterministic Lateral Displacement (DLD)

Enhanced the DLD theory for particle separation in real-life applications by developing computational models. This led to the creation of the "Boundary Correction Paradigm" to optimize sorting conditions for practical devices.

Density Functional Theory Calculations of H2S Detection on Strained MoS2

Performed density functional theory (DFT) calculations to explore the adsorption and sensing properties of H2S on strained and pristine single-layer MoS2, revealing critical insights into electron transfer and molecular physisorption.

Predicting Bioparticle Trajectories in Microfluidic Devices

Designed and implemented an efficient particle trajectory simulation algorithm for non-inertial microfluidics. This method accelerates simulations and accounts for particle-wall collisions, significantly improving runtime and accuracy.

van der Waals DFT Calculations of DNA on MoS2 and WS2

Utilized van der Waals density functional theory (vdW-DFT) to compare the adsorption characteristics of mutated, methylated, and canonical DNA bases on MoS2 and WS2, paving the way for advanced DNA sequencing technologies.

Early Career and Side Projects

Financial Risk Analysis Using Artificial Neural Networks

Developed a predictive model using Keras and TensorFlow to assess the financial risk of lending loans based on historical customer data. The ANN model achieved an F1-score of 0.89, providing valuable insights into borrower risk.

Heart Disease Prediction Using Logistic Regression

Built a logistic regression model to predict heart disease in patients based on medical data. Optimized to reduce false negatives, the model achieved an F1-score of 0.86, offering valuable predictive insights for early diagnosis.

Python Package for Gaussian and Binomial Distributions

Created a Python package for calculating probability density functions, generating histograms, and performing operations on Gaussian and Binomial distributions. The package allows users to easily handle statistical tasks and visualization.

Bank Note Authentication Using Random Forest Classifier

Applied a random forest classifier to authenticate banknotes based on image properties. The model was trained and tested with an F1-score of 0.99, proving highly effective in distinguishing real from counterfeit notes.

Classified Data Supervised Learning Model

Implemented a K-Nearest Neighbors (KNN) supervised learning model to classify datasets. The optimal K-value was determined, achieving an F1-score of 0.86, ensuring accurate classification with minimal error.

E-Commerce Review Sentiment Analysis Using NLP

Performed sentiment analysis on customer reviews from an e-commerce platform using natural language processing (NLP) techniques. Classified reviews into 1-star and 5-star categories, achieving an F1-score of 0.96.
